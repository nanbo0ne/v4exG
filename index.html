🚀 快速上手

按这个顺序走一遍，就能从项目设定开始生成大纲和正文。这里不会读取或展示你的默认提示词正文。

1. 配置 API

打开设置，填写 Base URL 和 API Key。点“获取模型”或手动输入模型名，然后保存。

2. 填写项目

在项目信息里填写书名、题材、世界设定、人物设定、章节数和单章目标字数。

3. 调整提示词

在写作提示词区按你的写法调整大纲、正文、去 AI、结尾提示词；也可以直接使用当前配置。

4. 设置创作规则

选择叙事视角，填写基调、禁止事项、写作风格约束；需要参考作品时，先导入参考文本并生成写作卡片、引用手法或大师洞察。

5. 生成大纲和正文

可以先手动生成大纲，再逐章生成正文；也可以点“一键生成”，让程序从大纲到章节连续完成。

6. 检查与导出

正文完成后可续写、写结尾、去 AI、检查一致性，最后在导出页导出 TXT、Markdown、HTML 或项目备份。
